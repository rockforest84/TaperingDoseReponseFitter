# HTML Port

This folder contains a browser-only port of the tapering dose response fitter.

## Open locally

Open [index.html](/c:/Users/peter/OneDrive/Python/TaperingDoseReponseFitter/html/index.html) directly in a browser.

If popup, clipboard, or download behavior is restricted when opening from `file://`, serve the folder statically instead. Any static file server will work.

## Deploy to GitHub Pages

Publish the `html/` folder as the site root.

## Features included

- Browser-side port of the PK, PD, calibration, schedule, and feasible-days logic
- Slider-based controls close to the desktop GUI
- CSV download
- Graph PNG download
- Popup print preview page with browser print
